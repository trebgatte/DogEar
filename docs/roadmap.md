# Roadmap

## Near term

- Improve turn extraction resilience
- Improve descriptor quality
- Reduce UI overhead further
- Add better empty/error states for local AI failures
- Add clear indication of heuristic vs AI-generated labels

## Medium term

- Descriptor modes:
  - Fast
  - AI
  - Manual
- Better caching and invalidation
- Per-conversation saved view state
- Better export options
- Optional turn grouping / sections

## Longer term

- Conversation outline mode
- Topic clustering mode
- More explicit debug panel
- Packaging for Chrome Web Store
- Better automated test coverage

## Non-goals for now

- remote SaaS backend
- hidden telemetry
- account system
- cloud-only summarization requirement
