# DogEar Roadmap

## v0.1.0 — Scaffold
- [x] Repository structure and shell files
- [ ] Basic Manifest V3 extension loads in Chrome/Edge

## v0.2.0 — Core Navigation
- [ ] Detect conversation turns in ChatGPT
- [ ] Render a collapsible sidebar with jump-to links
- [ ] Persist navigation state across page reloads

## v0.3.0 — Bookmarks & Labels
- [ ] Allow users to manually bookmark specific turns
- [ ] Support custom labels for bookmarks

- [ ] Integrate with locally running LLM (Ollama / LM Studio)

- [ ] Full browser compatibility (Chrome, Edge, Firefox)
- [ ] Polished UI with accessibility support
- [ ] Comprehensive documentation
- [ ] Published to Chrome Web Store

## Future Ideas
- Export conversation outlines as Markdown
- Shared bookmark sets (opt-in, no account required)
