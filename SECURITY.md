# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in DogEar, please do **not** open a public issue.

Instead, please report it privately by emailing the maintainer or using GitHub's private vulnerability reporting feature.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fixes if available

You can expect an acknowledgment within 48 hours and a resolution or mitigation plan within 14 days.
